# North Star

Completed audits

This is the main success metric because it directly measures whether users find enough value to finish the audit and view recommendations.

## Input metrics

- Landing page → form conversion
- Audit completion rate
- Email capture rate
- Share link clicks

## First instrumentation

- audit_started
- audit_submitted
- email_captured
- report_shared

## Pivot trigger

If audit completion stays below 20% after the first 100 users, simplify the form and reduce friction.
